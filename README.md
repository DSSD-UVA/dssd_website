# UVA Data Science for Sustainable Development (DSSD)

The official website for UVA's Data Science for Sustainable Development club. Built with React, TypeScript, and Vite, featuring an "Academic Brutalist" design system with bold typography, boxy geometry, and high-contrast editorial aesthetics.

**Live site:** [uvadssd.vercel.app](https://uvadssd.vercel.app)

## Getting Started

```bash
# Install dependencies
npm install

# Start the development server
npm run dev

# Build for production
npm run build

# Preview the production build
npm run preview
```

## Tech Stack

- React + TypeScript
- Vite
- Vercel (hosting)
